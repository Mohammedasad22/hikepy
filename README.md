# Prerquests

1. Android sdk
2. java

# Setup

1. Go to https://appium.io/
2. Download and install appium for your os
3. Start the server

# Clone and change directory

```
git clone https://github.com/eternalodball/hikepy
cd hikepy
```

# Install requirements
```
pip3 install -r requirements.txt
```
# Run it !

```
python3 main.py
```
# Note -
